# Subspace Emissary EX

**Subspace Emissary EX** is a framework created by Kapedani that expands the ability to modify Brawl's Subspace Emissary game mode. This system allows the following, plus more:

- Add new fighters to SSE, including making them unlockable with customizable requirements
- Add new doors to SSE, including doors that can take you to whole new locations, and even with customizable requirements (score requirements, random, etc)
- Support for custom cutscenes
- Time attack mode (hold X when selecting a fully cleared level)
- Speedrun timer (hold Y when selecting a level)
- Select locked characters (hold L when selecting a difficulty)
- Select the same character multiple times (hold Y when trying to hover over the character)

This system is included in many modern EX builds.
