# Soundbank Expansion System

The **Soundbank Expansion System** is a framework created by JOJI to add more available [soundbanks](soundbanks.md) to the game. This system adds **256** new available soundbanks, starting at ID 331 and ending at ID 586.

Custom soundbanks are all based on Snake's soundbank, meaning to generate a custom soundbank you only need to use Snake's soundbank as a base and export it with your preferred custom soundbank ID.

Most modern builds integrate this system.
