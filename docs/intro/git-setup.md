# Git

If you are managing a build or modding long-term, it can be extremely useful to set up a **git** repository. Git is a source control tool that allows you to set up a repository where all changes are tracked. Setting up a Brawl build as a git repo allows you to revert changes you make when you run into issues, making it a very useful tool when you're making lots of changes over time.

To learn more about setting up a Git repo for your build, see [Mari0shi's guide](https://mari0shi.github.io/tutorials/brawlinstaller/git/) on the subject. It is focused on dong so with BrawlInstaller, but you can skip those sections if they are not relevant to you.
