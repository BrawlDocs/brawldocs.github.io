# Community

Still have questions? Or maybe you just want to talk to some like-minded people? Either way, this will serve as a list of places you can go to get involved in the Brawl modding community.

- [Custom Brawl Modding](https://discord.gg/GbxJhbv) - One of the main Discord communities for modding Brawl, this is a general-purpose server all about modding the game.
- [Project+ Discord Server](https://discord.com/invite/vdssRDg) - The official Discord server for [Project+](/intro/initial-setup?id=project).
- [PROJECT REX Discord Server](https://discord.com/invite/vegZup9xVc) - The official Discord server for [REX](/intro/initial-setup?id=rex).
- [Brawl Modding Workshop](https://discord.gg/4NPt7RM) - A Discord server where users can create their own workshops to show off their modding work for Brawl.
- [BrawlInstaller Discord Server](https://discord.gg/NMkTZWt9bK) - The official Discord server for [BrawlInstaller](/intro/tools?id=brawlinstaller).
- [Brawl Knowledge Compendium](https://discord.gg/s7c8763) - The official Discord server for [BrawlCrate](/intro/tools?id=brawlcrate).
