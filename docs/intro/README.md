# Getting Started

This is a repository of knowledge on Brawl modding, intended to be a starting point for finding resources all in one place.

This repository is not comprehensive. It is a knowledge base with basic information and links to various, more detailed resources and guides, so they can all be found in one spot, even if they are not all hosted here. The content of the site will also be updated over time.

This section will help you get started on modding Brawl by going over the basics of setting up a modded build, as well as providing links to resources that can help you dive deeper into modding the game.
