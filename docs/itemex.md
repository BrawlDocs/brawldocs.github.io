# ItemEx

**ItemEx** is a system by Kapedani that expands item functionality greatly, adding many new features to items and also making it much easier to add new items to the game, whether they are global items, items generated by fighters, or items generated by stages.

ItemEx allows:
- Common items to be overridden on a per stage basis
- Stages to have more control over items
- Fighter specific item clones
- Cloned Pokemon/Assist variants
- Ability to adjust Pokemon/Assist probability per stage
- Any item to be able to have collision

ItemEx is used by builds like [Project+](getting-started?id=project) to add new items, such as the Double Cherry item, and is also used in builds like [P+Ex](getting-started?id=pex) and [REX](getting-started?id=rex) to add complex projectiles to characters that wouldn't otherwise be able to use them.
