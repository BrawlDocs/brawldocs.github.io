# BrawlDocs
A Brawl modding knowledge repository.

Uses [docsify](https://docsify.js.org/#/).

# Contributing

If you have information, such as guides or resources, that you think should be added, please feel free to open an issue with your suggestion and a link to any relevant resources.

If you would like to contribute directly, feel free to fork this repository, make changes, and create a pull request.

To test, open the repo folder in command line and run `docsify serve ./docs`. To deploy, in `Pages` in the repository settings, set the deploy branch to `master` using the `/docs` folder and it will deploy automatically whenever a change is pushed to master. The page can then be previewed from a link in the `Pages` section of the repository settings.
